# Static Assets

Dynamically generated visualizations from the `data_vizualization_tool`.

## Purpose

When users request visualizations:
1. Charts are generated using matplotlib/plotly
2. Images saved to this directory
3. Served through the Streamlit interface

## Contents

- Bar charts, line plots, pie charts, scatter plots
- Heatmaps, distribution plots, trend analyses
- PNG/SVG format images

## Note

Files are temporary and regenerated on-demand.
