# Evaluation

Evaluation dataset and methodology documentation.

## Contents

### `Evaluation_dataset.ipynb`
Sanitized notebook showing the evaluation question generation process. Original 99+ evaluation questions removed for privacy.

### `EVALUATION_METHODOLOGY.md`
Comprehensive documentation of the evaluation framework:
- 99-question structure (33 easy, 33 medium, 33 hard)
- Question generation process
- Difficulty level definitions
- Coverage matrix
- Out-of-scope question handling

## Evaluation Approach

The methodology documents a systematic evaluation covering:
- **Easy**: Simple filtering and aggregation
- **Medium**: Multi-condition queries with grouping
- **Hard**: Complex multi-step analysis
- **Out-of-Scope**: System limitation recognition

## Note

The original evaluation dataset contained company-specific questions and has been removed. The methodology and framework are preserved in the documentation.
