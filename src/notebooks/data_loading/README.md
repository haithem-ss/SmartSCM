# Data Loading

Notebooks for prototyping and testing data loading functionality.

## Contents

### `index.ipynb`
Development notebook for the `data_loader_tool`. Tests date-range filtering, file loading, and DataFrame concatenation.

## Purpose

Prototyped the data loading logic that became the production `data_loader_tool.py` in `/src/tools/`.

## Usage

The production tool is used by agents to load data between specified dates from `/src/final_data/data/`.
