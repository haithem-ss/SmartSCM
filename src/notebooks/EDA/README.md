# Exploratory Data Analysis (EDA)

Notebooks for exploring and analyzing the order dataset.

## What's Inside

EDA notebooks exploring:
- Data distributions and patterns
- Customer ordering behavior
- Product popularity trends
- Regional sales analysis
- Order status and delivery metrics

## Dataset

Analysis uses the synthetic order dataset from `/src/final_data/`:
- Daily order files (2025-01-15 through 2025-01-29)
- Merged dataset with customer, product, and order information
- Schema documented in `data_documentation.yaml`
